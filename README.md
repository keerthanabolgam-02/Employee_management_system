# Employee Management System

## Description
The Employee Management System is a Python-based application designed to manage employee records efficiently. It allows users to add, update, delete, search, and display employee information using a simple interface.

## Features
- Add Employee
- View Employee Details
- Update Employee Information
- Delete Employee Records
- Search Employees
- Store Data Using CSV Files

## Technologies Used
- Python
- Pandas
- CSV File Handling
